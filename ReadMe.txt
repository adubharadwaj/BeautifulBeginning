How to access the web home page:

Step 1: Please unzip & copy AmericanEagle folder to your personalized folder.

Step 2: Open command prompt and navigate to that folder path and type "npm start". It will automatically install packages needed.

Step 3: Open any browser and type "http://localhost:8000/home#/home". It will navigate to the webpage.

Step 4: To see the navigation go to "Christening" tab and click on "Silver". The page will display "Silver" in that page since the webpage is made as single application page.  

Note: All the other navigation links are dummy. Can be easily changed if you add the page related details in "app.js" file.
All the script tags are addded below in the body tag to make sure the webpage loads faster.
